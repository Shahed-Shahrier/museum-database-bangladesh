Unified Museum Database for Bangladesh

This repository includes a PHP + MySQL web application. GitHub Pages can host only static files, so this page is the project landing page. The full application is deployed using container/server workflows.

Repository

github.com/Shahed-Shahrier/museum-database-bangladesh

Deployment Notes

  • GitHub Pages: static landing page (this site)
  • CI lint workflow: checks PHP syntax
  • GHCR workflow: builds and publishes container image
  • Server deploy workflow: deploys full app via Docker Compose + SQL updates

Next Step
